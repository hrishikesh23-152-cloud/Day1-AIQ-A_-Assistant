# 🚀 Day1 AIQ Assistant

My first GenAI project built with **Next.js** and **Node.js**.

🌐 **Live Demo:** https://day1-aiq-a-assistant-client.onrender.com/

---

## 📖 About

Day1 AIQ Assistant is a simple AI-powered question-answering application. Users can enter a prompt, send it to the backend, and receive an AI-generated response.

This project marks the beginning of my GenAI development journey and helped me understand how frontend applications communicate with AI-powered backend services.

---

## ✨ Features

- 🤖 AI-powered question answering
- ⚡ Fast and responsive UI
- 🔗 Frontend & backend API integration
- 📝 Structured AI responses
- 📱 Responsive design
- ☁️ Deployed on Render
